# Sharing Information on LAN using Cisco Packet Tracer

## Overview
This project demonstrates communication and information sharing within a Local Area Network (LAN) using Cisco Packet Tracer.

## Objectives
- Understand LAN communication
- Configure devices in the same network
- Test connectivity between systems
- Share information across connected devices

## Tools Used
- Cisco Packet Tracer

## Devices Used
- PCs
- Switch
- Ethernet Cables

## Features
- LAN setup
- IP addressing
- Connectivity testing using ping command
- Information sharing between devices

## Learning Outcomes
Users will learn:
- LAN configuration
- Basic IP addressing
- Device communication within the same network

## Conclusion
This project helps in understanding how Local Area Networks work and how devices communicate and share information within a network.
